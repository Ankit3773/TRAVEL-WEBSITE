Trip Standards

What passengers can expect during the journey

Before Boarding

  • Choose routes using a dedicated results page
  • Compare departure windows and fare ranges
  • Review route type before seat selection

At Boarding

  • Carry booking confirmation and a valid ID
  • Arrive early enough to settle luggage and seats
  • Use support contact if boarding coordination is needed

During Travel

  • Travel history and confirmation remain accessible
  • Payment mode is visible in the final booking summary
  • Seat and trip details stay clear for every booking
